DROP YOUR RAW RESUMES, LINKEDIN LINKS, OR PORTFOLIO LINKS HERE!

Career-Ops is an AI-powered engine. If this is your first time using it, you DO NOT need to manually configure the profile files.

Simply paste your old raw resume text into a file in this folder, or drop a text file with a link to your LinkedIn/Portfolio. 

Once you provide some context in this folder, just run any Career-Ops command (like a job URL) in the CLI. The AI will read the files in this folder, ask you a few clarifying questions if needed, and then automatically build your `cv.md` and `config/profile.yml` for you!
