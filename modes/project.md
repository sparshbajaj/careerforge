# Mode: project — Portfolio Project Evaluation

Evaluate a personal project or portfolio piece against the current market requirements or a specific job role.

## Workflow

1. **Select Project**: Paste project description, README, or URL.
2. **Analysis**:
   - Identify "Signal": What does this project prove? (Technical depth, AI usage, product sense).
   - Tech Stack: List languages, frameworks, and AI models used.
   - Impact: Highlight quantified results if available.
3. **Market Alignment**: How does this project help landing the target archetypes?

## Output Sections

- **Proof Points**: Bullet points ready for the CV.
- **STAR Story**: Draft a STAR story for this project.
- **Improvement Plan**: "To make this project 10x more valuable for [Archetype], add X."
- **Article Digest Addition**: Draft a compact version for `article-digest.md`.
